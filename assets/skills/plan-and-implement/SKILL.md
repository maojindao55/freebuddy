# Plan and implement

Clarify the intended outcome from available context, inspect the current implementation, then execute the smallest coherent change.

- State assumptions that materially affect scope.
- Preserve existing interfaces unless the request requires changing them.
- Implement end to end, including persistence and UI wiring where relevant.
- Verify the result and summarize the user-visible behavior.
